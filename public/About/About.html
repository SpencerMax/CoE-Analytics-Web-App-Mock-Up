Project Mission

Hello everyone! We are the web analytics team for P.C. Rossin College of Engineering and Applied Science at Lehigh University. Our mission is to develop analytical tools such as dashboards and custom reports that will help the marketing team be more successful when making campaigns and advertisements to provide value for our stakeholders. We are currently creating a Web application to make it easier for our marketing team to access and interpret data.

Technolgies/Tools

1) Node.js framwork for the backend

2) Html, CSS, and Javascript for the frontend

3) Google Analytics Reporting API to fetch data from Google

4) Google Analytics Tools (i.e. dashboards and custom reports)

Current Challenges

Google is currently in the process of ending support for Universal Analytics, which is how the College of Engineering is currently collecting data, and replacing it with a different method called GA4. This has caused issues for us as some APIs, such as the Analytics Management API, are different for the two. We have addressed this issue by working to ensure our code works with both collection methods. Another minor issue we have run into is that certain APIs require administrator access. This took some time to acquire, but once that hurdle was cleared the problem was solved.
